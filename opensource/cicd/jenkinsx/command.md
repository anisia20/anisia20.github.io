### command
    jx admin operator : 최초 jenkins x 설치
    jx ns jx : jx namespace 로 변경 
    jx admin log : jenkins x 의 작업 로그를 확인 가능
    jx get build log : pipeline log를 선택하여 확인 가능
    jx get application : jenkins x 에서 구동한 파드 확인 가능
    jx get activities : jenkinx x 에서 작업 처리 순 성공여부 확인 가능
    jx import : 외부에서 소스 가져와 jenkins x 포멧으로 만들어줌
    jx create quickstart : 샘플 소스 만들기
    jx secret verify : secret 정보 확인
    jx edit secret : secret 정보 변경
